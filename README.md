# Wearable Device IMU Data Collection

This repository provides software and code to read Sensor data from wearable devices (smartphones, smartwatches, earphones) across three platforms: Android, Huawei, and Apple.

## Introduction

This project enables the collection of IMU data from everyday wearable devices like smartphones and smartwatches. It covers three platforms: Android, Huawei, and Apple, each with specific implementation details for accessing sensor data.

## System Overview

### Android
- Android smartphones and smartwatches.

### Huawei
- Huawei smartphones, smartwatches, and earphones.

### Apple
- Apple smartphones, smartwatches, and earphones.

## Installation
### Android
1. **For Smartphones**:
   - Download the **PhoneIMU.apk** from [this link](https://cloud.tsinghua.edu.cn/f/6133d04747394d298bb5/?dl=1).
   - Install the APK file on your Android smartphone.

2. **For Smartwatches** (if applicable):
   - Download the **WatchIMU.apk** from [this link](https://cloud.tsinghua.edu.cn/f/69fd969c4ab24b5da0dd/?dl=1).
   - Install the APK file on your smartwatch.

### Livedemo
We use Unity to visualize the live demo. To get started:
1. Download the **Mocap Live Demo Unity package** from [this link](https://cloud.tsinghua.edu.cn/f/c1fded497d7f441793e0/?dl=1).
2. Download and install the **Unity Editor version 2022.3.20f1c1** from the [Unity Hub](https://unity.com/download).
3. Open the downloaded project in the Unity Editor.
4. In the Unity Editor, go to the **Scenes** folder and load the **Motion Viewer** scene.
